📺 大屏看板使用说明

写给产品团队的零代码维护指南 — 你只需要会改JSON文件就行

📁 文件说明

llm-dashboard/
├── index.html     ← 看板主页面（不需要改）
├── data/
│   └── dashboard.json   ← ⭐ 你唯一需要改的文件
└── 使用说明.html   ← 你正在看的这个
核心原则：你只需要修改 data/dashboard.json 这一个文件。修改后，刷新浏览器就能看到更新。

🚀 怎么打开看板

1双击 index.html 文件，浏览器会自动打开

2按 F 键进入全屏模式

3连接电视投屏（HDMI或无线投屏都可以）

快捷键：
← → 手动切换页面   Space 暂停/继续自动轮播   F 全屏/退出全屏

✏️ 怎么更新数据

用任意文本编辑器（推荐 VS Code，也可以用记事本）打开 data/dashboard.json，修改对应内容，保存后刷新浏览器即可。

注意：JSON格式很严格，漏一个逗号就会出错。修改前建议先复制一份备份。如果改坏了，可以用备份恢复。

1️⃣ 更新项目进展数据

找到 "project" 下面的 "summary"，修改对应数字：

"summary": {
  "totalTasks": 24,        ← 总任务数
  "completed": 15,         ← 已完成数
  "inProgress": 6,         ← 进行中数
  "blocked": 1,            ← 阻塞数
  "notStarted": 2          ← 未开始数
}

2️⃣ 更新里程碑

找到 "milestones" 数组，每个里程碑的格式：

{
  "name": "Prompt工程优化",     ← 里程碑名称
  "status": "inProgress",       ← 状态（见下表）
  "date": "2026-03-30",          ← 预计完成日期
  "progress": 65                ← 进度百分比（0-100）
}
status 值含义看板显示
"completed"已完成✅ 绿色标签
"inProgress"进行中🔵 蓝色进度条
"notStarted"未开始⚪ 灰色

3️⃣ 更新评测数据

找到 "evaluation" 部分：

"evaluation": {
  "currentModel": "内部模型 v2.3",   ← 当前模型名
  "lastEvalDate": "2026-03-20",      ← 上次评测日期
  "overallScore": 78.5,              ← 综合得分
  "previousScore": 72.1,             ← 上一版得分（用于计算变化）
  ...
}

更新各维度得分（雷达图会自动变化）：

"dimensions": [
  { "name": "多轮对话", "score": 82, "previousScore": 75, "fullScore": 100 },
  { "name": "知识问答", "score": 85, "previousScore": 80, "fullScore": 100 },
  ... 直接改 score 数字就行，雷达图自动更新
]

更新准召率数据：

"precisionRecall": [
  { "scenario": "客服问答", "precision": 89.2, "recall": 83.5 },
  ... precision = 准确率, recall = 召回率
]

新增一个版本的得分（趋势图会多一个点）：

"versionTrend": [
  ... 已有的版本不要删 ...
  { "version": "v2.5", "score": 82.0, "date": "2026-04" }  ← 新加这行
]

4️⃣ 更新团队成员 & 文档

找到 "members" 数组，每个成员的格式：

{
  "name": "张三",
  "role": "产品负责人",
  "currentTask": "多轮对话能力PRD",       ← 当前在做什么
  "status": "inProgress",               ← 状态
  "progress": 80,                         ← 进度百分比
  "documents": [
    {
      "title": "多轮对话能力需求PRD",     ← 文档标题
      "url": "https://doc.weixin.qq.com/xxx",  ← 企微文档链接
      "type": "prd",                      ← 类型：prd / report
      "updateDate": "2026-03-22"           ← 更新日期
    }
  ]
}
添加新成员：复制一个已有成员的整段，粘贴在最后一个 } 后面（记得加逗号），然后修改内容。
添加新文档：在某个成员的 "documents" 数组里，复制一条已有的，粘贴后修改。

5️⃣ 修改看板配置

"config": {
  "title": "大模型能力建设项目",    ← 看板标题
  "subtitle": "项目作战大屏",       ← 副标题
  "autoPlay": true,                 ← 是否自动轮播（true/false）
  "autoPlayInterval": 30,           ← 自动切换间隔（秒）
  "theme": "dark"                   ← 主题（目前只有dark）
}

🔧 常见问题

Q: 改了JSON后看板没变化？

按 Ctrl+Shift+R（Mac: Cmd+Shift+R）强制刷新浏览器。

Q: 页面一片空白？

JSON格式写错了。常见原因：少了逗号、多了逗号、引号不配对。用 jsonlint.com 检查一下你的JSON格式。

Q: 怎么加新的评测维度？

在 "dimensions" 数组里加一条新的就行，雷达图会自动多一个角。

Q: 怎么加新的场景准召数据？

在 "precisionRecall" 数组里加一条：{ "scenario": "新场景", "precision": 85.0, "recall": 80.0 }

Q: 企微文档链接怎么获取？

在企微文档右上角点「分享」→「复制链接」，粘贴到JSON的 "url" 里即可。点击卡片上的文档会在新窗口打开。
